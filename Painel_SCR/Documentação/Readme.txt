Projeto de Criação da DW contendo os dados Sistema Central de Risco do Banco Central do Brasil
Os dados de empréstimos e financiamentos registrados no Sistema Financeiro Nacional foram normalizados e organizados em uma
DW no SGBD MariaDb. Foi criado um dashboard no Microsoft PowerBi para facilitar a Análise dos Dados.
São analisados dados de 2023 e 2024 (até Março).
